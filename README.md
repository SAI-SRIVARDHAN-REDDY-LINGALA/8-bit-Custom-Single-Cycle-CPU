# 8-bit-Custom-Single-Cycle-CPU
Custom 8-bit single-cycle CPU built in Verilog with Harvard architecture, separate instruction/data memory, 4 registers, and 8 instructions (HALT, LOAD, STORE, ADD, SUB, ADDI, JMP, AND). Fully verified with testbench, GTKWave waveform tracing, and documented control logic — a stepping stone toward a RISC-V core.
